escapes

escapes.js is a small JavaScript library for rendering ANSI art in the browser. It has a simple API, designed to encourage experimentation. It has no dependencies, but it works well with jQuery.

        escapes('static/TT-SOH.ICE', function () {
          $(this).appendTo('body');
        });

usage

escapes( url, callback )

Within the callback, the keyword this refers to a <canvas> element on which the ANSI was painted. The canvas object (not the prototype!) is augmented with two additional methods:

canvas.toImageTag()
returns an <img> element of the rendered ANSI.
canvas.toDownloadURL()
returns a URL for downloading (saving) the image.

If you're using jQuery, you can also invoke escapes as a jQuery plugin: 

$.ansiRender( url, [callback] )

This returns a jQuery.Deferred object.
